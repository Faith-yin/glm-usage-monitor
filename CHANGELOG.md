# 版本日志

## v1.0.1

Release title: **v1.0.1 — 体验优化与 Bug 修复**

### ✨ 优化

- 刷新间隔选项调整为 1 分钟 / 5 分钟 / 10 分钟 / 30 分钟 / 1 小时 / 2 小时，默认 1 小时
- Tooltip 卡片支持圆角（Windows 11），间距和尺寸优化
- 重置时间改用英文缩写（3h 5min、2d 3h），日期附带星期（06-20 周六 18:02）
- API Key 弹窗：DPI 适配，自动回填已保存 Key
- 获取链接可点击直接跳转浏览器

### 🐛 修复

- 托盘图标偶尔持续显示问号的问题
- 右键菜单展开时 Tooltip 意外弹出的问题
- 鼠标在图标区域内移动时 Tooltip 反复闪烁的问题

### 📥 下载

下载 `GLM用量监控.exe` 后双击运行（或覆盖旧版），右键托盘图标设置 API Key 即可使用。

## v1.0.0

Release title: **🎉 v1.0.0 首次发布**

### ✨ 功能

- 系统托盘实时监控 GLM Coding Plan 用量配额
- 悬停托盘图标弹出极简卡片，显示 5 小时 / 每周 / MCP 每月三项额度
- 托盘图标颜色指示：绿(<50%) / 橙(50~79%) / 红(≥80%)
- 自动刷新 + 缓存兜底机制
- 可配置刷新间隔（1/3/5/10/15/30 分钟）
- 开机自启支持
- 详情窗口手动刷新
- 单实例运行保护

### 📥 下载

下载 `GLM用量监控.exe` 后双击运行，右键托盘图标设置 API Key 即可使用。
