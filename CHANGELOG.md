# 版本日志

## v1.0.1

Release title: **v1.0.1 — 体验优化与 Bug 修复**

### 🐛 Bug 修复

- **修复托盘图标始终显示问号**：`from api import _current_data` 绑定导入在数据刷新后引用失效，改为就地更新 + 模块直读，图标颜色恢复
- **修复 API 调试日志静默丢失**：`api.py` 补齐缺失的 `import os`，网络异常时调试信息不再被吞
- **修复 TooltipWindow 双重实例**：移除 `widgets.py` 中无用的全局单例，消除死代码
- **修复右键菜单显示时 tooltip 弹出**：新增 `_menu_visible` 标记，菜单生命周期内屏蔽悬停进入
- **修复鼠标在托盘按钮区域内移动时 tooltip 反复闪烁**：离开检测的锚点随 `WM_MOUSEMOVE` 实时更新，不再误判

### ✨ 新增 / 优化

- **刷新间隔选项调整**：改为 1 分钟 / 5 分钟 / 10 分钟 / 30 分钟 / 1 小时 / 2 小时，默认 1 小时
- **Tooltip 圆角**：通过 Windows 11 DWM API 实现原生圆角效果
- **Tooltip 视觉调整**：距光标 50px，左右内边距 28px，顶部 24px，底部 14px，宽度 280
- **重置时间格式优化**：时间单位改为英文缩写（3h 5min、2d 3h），末尾保留"后重置"
- **重置日期新增星期**：非当天显示 `06-20 周六 18:02` 格式
- **API Key 弹窗优化**：DPI 自适应尺寸，打开时间填已有 Key（星号遮盖），获取链接改为 `https://open.bigmodel.cn/apikey/platform` 且可点击跳转
- **开发热重载**：新增 `dev.py`，监听源码变更自动重启，无需手动停启

### 🔧 工程改进

- **线程安全**：`_current_data` 写入加锁，`_do_refresh_inner` 返回刷新结果
- **消除重复逻辑**：tray.py 悬停刷新复用 `_do_refresh_inner`，不再手写 fetch/parse/update 流程
- 移除 `save_config(load_config())` 等无意义调用

### 下载

下载 `GLM用量监控.exe` 后双击运行（或覆盖旧版），右键托盘图标设置 API Key 即可使用。

## v1.0.0

Release title: **🎉 v1.0.0 首次发布**

### 功能

- 系统托盘实时监控 GLM Coding Plan 用量配额
- 悬停托盘图标弹出极简卡片，显示 5 小时 / 每周 / MCP 每月三项额度
- 托盘图标颜色指示：绿(<50%) / 橙(50~79%) / 红(≥80%)
- 自动刷新 + 缓存兜底机制
- 可配置刷新间隔（1/3/5/10/15/30 分钟）
- 开机自启支持
- 详情窗口手动刷新
- 单实例运行保护

### 下载

下载 `GLM用量监控.exe` 后双击运行，右键托盘图标设置 API Key 即可使用。
